# Сторонние компоненты

FreeMon использует следующие открытые компоненты. Они принадлежат их авторам
и распространяются на условиях их лицензий.

## LibreHardwareMonitorLib
- Назначение: чтение датчиков железа (температуры, нагрузка, частоты, мощность, вентиляторы).
- Сайт: https://github.com/LibreHardwareMonitor/LibreHardwareMonitor
- Лицензия: Mozilla Public License 2.0 (MPL-2.0)

## PresentMon
- Назначение: измерение FPS / времени кадра (через ETW).
- Сайт: https://github.com/GameTechDev/PresentMon
- Лицензия: MIT License
- Примечание: PresentMon.exe — отдельная программа Intel, поставляется как есть.
  При распространении сохраняйте этот файл лицензий.

## .NET / Microsoft
- Среда выполнения .NET и сопутствующие библиотеки Microsoft.
- Лицензия: MIT License
